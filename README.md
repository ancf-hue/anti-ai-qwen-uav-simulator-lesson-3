# 🐱 Anti-AI Qwen БПЛА Тренажёр — Урок 3: Ночная охота

**Бесплатный социальный офлайн-3D-тренажёр для подготовки операторов БПЛА**  
*Проект образовательной платформы [Anti-AI Qwen Academy](https://ancf-hue.github.io/)*

---

## 📖 О проекте

Третий урок серии тренажёров для операторов БПЛА. Оператор с позывным **"КОТ"** 🐱 должен в условиях ночной скрытной охоты бесшумно подкрасться к 5 целям и захватить их, удерживая прицел 1.5 секунды на каждой.

### 🎯 Учебные цели Урока 3:
- Отработка навыков скрытного пилотирования в ночных условиях
- Управление уровнем шума дрона (зависит от скорости)
- Работа с тепловизором для обнаружения целей
- Тренировка терпения и точности (захват удержанием, а не кликом)
- Распределение времени (миссия ограничена 3 минутами)

---

## ✨ Возможности

| Функция | Описание |
|---|---|
| 🌙 **Ночной режим** | Тёмная сцена с плотным туманом, лунным светом, звёздами и силуэтами деревьев |
| 🔦 **Прожектор дрона** | Дрон освещает путь в темноте — единственное яркое пятно |
| 🐭 **Цели-"мыши"** | 5 быстрых мишеней, которые бегают по земле и меняют направление |
| 🔊 **Индикатор шума** | Зависит от скорости дрона. При >60% мыши убегают! |
| 🎯 **Захват удержанием** | Нужно держать ЛКМ на цели 1.5 секунды (не просто кликнуть) |
| 🌡️ **Тепловизор (T)** | Переключает режим — мыши подсвечиваются оранжевым |
| 😿 **Пугливые мыши** | Если дрон шумный и близко — мыши убегают в противоположную сторону |
| ⏱️ **Таймер миссии** | 3 минуты на выполнение задания |
| 💜 **Дрон КОТ** | Фиолетовая подсветка, красная FPV-камера, фиолетовый след |
| 🚪 **Кнопки выхода** | Возврат в меню и перезапуск в любой момент |
| 📱 **Адаптивность** | Работает на ПК и мобильных устройствах |
| 🌐 **Офлайн-режим** | Не требует интернета (кроме первой загрузки Three.js) |

---

## 🎮 Управление

| Клавиша | Действие |
|---|---|
| `W` / `↑` / `Ц` | Газ (вперёд) |
| `S` / `↓` / `Ы` | Тормоз / Задний ход |
| `A` / `←` / `Ф` | Поворот влево |
| `D` / `→` / `В` | Поворот вправо |
| **ЛКМ (удерживать)** | Захват цели (прицел в зоне 1.5 сек) |
| `T` | Переключение тепловизора 🌡️ |
| `Пробел` | Стабилизация дрона |
